# 🛍 Product List App (React Native + TypeScript)

A simple and modern **Product Listing Mobile Application** built using **React Native & TypeScript**.
This app displays products with images, allows searching, and provides a clean user interface.

---

## 🚀 Features

* 📦 Display product list (4+ products)
* 🔍 Search functionality
* 🖼 Product images (real & working URLs)
* 📱 Clean and responsive UI
* ⚡ Fast and lightweight app

---

## 🛠 Tech Stack

* React Native
* TypeScript
* React Navigation
* Redux Toolkit (optional version)
* AsyncStorage (for saving products)

---

## 📸 Screens

* Product List Screen
* Product Detail Screen
* Saved Products Screen

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/product-list-app.git
```

2. Install dependencies

```
npm install
```

3. Run the app

```
npx expo start
```

---

## 📌 Usage

* Open the app
* Browse product list
* Use search bar to filter products
* Click on a product to see details
* Save products (optional feature)
